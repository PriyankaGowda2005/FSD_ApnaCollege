h1 Heading

The <h1> tag is the most important heading. It is usually used for the main title of a webpage.

h2 Heading

The <h2> tag is used for sub-headings under h1.

h3 Heading

The <h3> tag is used for smaller sub-sections.

h4 Heading

The <h4> tag is used for less important headings.

h5 Heading

The <h5> tag is rarely used and represents minor headings.

h6 Heading

The <h6> tag is the smallest heading.

The <hr> tag creates a horizontal line and is used to separate content.

The <p> tag is used to write paragraphs of text. Browsers automatically add space before and after paragraphs.

The <br> tag is used to break a line
without starting a new paragraph.

The <b> tag makes text bold without giving it importance.

The <strong> tag makes text bold and important.

The <i> tag makes text italic for styling.

The <em> tag emphasizes text with meaning.

The <u> tag underlines text.

The <mark> tag highlights text.

The <small> tag shows smaller text.

The <sup> tag is used for superscript like 102.

The <sub> tag is used for subscript like H2O.

The <del> tag shows deleted text.

The <ins> tag shows inserted text.

The <a> tag is used for links: Visit Google

The <img> tag displays an image. The alt attribute provides alternative text if the image fails to load.

The <ul> tag creates an unordered list:

  • HTML
  • CSS
  • JavaScript

The <ol> tag creates an ordered list:

  1. Frontend
  2. Backend
  3. Database

The <table> tag is used to create tables.
