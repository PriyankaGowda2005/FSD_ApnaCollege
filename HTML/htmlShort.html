HTML Complete Guide (Beginner to Advanced)

HTML stands for HyperText Markup Language. It is used to create the structure of web pages. HTML is not a programming language — it is a markup language.

HTML works using tags. Tags tell the browser how to display content.

Basic HTML Structure

Every HTML document starts with <!DOCTYPE html>. It tells the browser that this document is written in HTML5.

The <html> tag is the root of the HTML document.

The <head> tag contains meta information like title and character set.

The <body> tag contains everything that is visible on the webpage.

Headings (h1 to h6)

h1 Heading

<h1> is the most important heading. Usually used once per page for the main title.

h2 Heading

<h2> is used for main sections under h1.

h3 Heading

<h3> is used for sub-sections.

h4 Heading

<h4> is used for smaller headings.

h5 Heading

<h5> is rarely used.

h6 Heading

<h6> is the smallest heading.

Text Formatting Tags

The <p> tag is used to write paragraphs.

The <b> tag makes text bold.

The <strong> tag makes text bold and important.

The <i> tag makes text italic.

The <em> tag emphasizes text with meaning.

The <u> tag underlines text.

The <mark> tag highlights text.

The <small> tag shows smaller text.

Superscript example: 102 using <sup>

Subscript example: H2O using <sub>

The <del> tag shows deleted text.

The <ins> tag shows inserted text.

Line Break and Horizontal Line

The <br> tag breaks the line
without starting a new paragraph.

The <hr> tag creates a horizontal line.

Links (Anchor Tag)

The <a> tag is used to create links.

Example: Open Google

The target="_blank" opens the link in a new tab.

Images

The <img> tag is used to display images.

The alt attribute provides text if the image fails to load. It is important for accessibility.

Forms

Forms are used to collect user input.

The <input> tag is used for text input.

The <input type='email'> is used for email.

The <button> tag submits the form.

Semantic HTML (Advanced)

Semantic tags give meaning to the content. They improve SEO and readability.

Examples of semantic tags: <header>, <nav>, <section>, <article>, <footer>

Conclusion

You have learned HTML from basic structure to advanced concepts. This is the foundation of web development.
